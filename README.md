# 🤖 AI FAQ Chatbot

An interactive FAQ Chatbot built using Python and Streamlit.

This chatbot answers user questions by matching them with the most relevant FAQ using Natural Language Processing (NLP) techniques.

---

## 🚀 Features

* Modern Chat UI
* FAQ-based intelligent responses
* Typing animation
* Enter key + Send button support
* Chat history
* Similarity-based answer matching
* Responsive design

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-learn
* TF-IDF Vectorizer
* Cosine Similarity

## ⚙ Installation

Clone repository:

git clone YOUR_REPOSITORY_LINK

Move to project folder:

cd CodeAlpha_FAQ_Chatbot

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py
